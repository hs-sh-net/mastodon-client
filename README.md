# Mastodonのクライアントです

## つかいかた:
> select <HTL|LTL|FTL>
> t <トゥート>
> re <ID> <返信>
> fav <ID>
> bt <ID>
> set vis <direct|private|unlisted|public>
> pause
> help

## Installation

#### Require
- NodeJS

```
git clone
cd
cp config.sample.js config.js
vi config.js # トークンなど

npm install
node main.js
```
